Hangman

Rules

  • Each turn you choose a letter (tap / click or use keyboard)
  • If that letter exists in the mystery word, it will be shown
  • If that letter is not part of the word, then a new body part will be added
  • If all body parts are added then you are officially hanged (anyone else hate seeing or hearing it this way?)
  • Tip: There are 7 body parts, so 7 fails and you lose...
